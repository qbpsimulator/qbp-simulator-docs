complexType DistributionInfo

Documentation
                Distribution info type is used to define durations of
                elements following given distributions. All values are
                in seconds.
Properties
This component is not nillable.
Model
<...
  arg1 = double

</...>

Nested Element Summary
 exactValues
           List exact numeric values to be used as the distribution.  
 histogramDataBins
          List of histogram bins to define distributions with probabilities. 
 timeUnit
          Does not actually specify time unit - values must be given always in seconds. 

Attribute Summary
 doublearg1
           For distributions: uniform - minimum value; exponential - mean, will be used to calculate 1/mean for exponential distribution; gamma - variance; triangular - minimum at "a" parameter; lognormal - variance;  
 doublearg2
           For distributions: uniform - maximum value; triangular - maximum at "b" parameter;  
 doublemean
          For fixed, normal, gamma, triangular and lognormal distributions. 
 qbp:DistributionTypetype
          Distribution type
